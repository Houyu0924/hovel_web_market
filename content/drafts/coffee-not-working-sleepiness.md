# コーヒーが効かなくなった人へ｜眠気をごまかす前に見直すべきこと

朝にコーヒーを飲んでも目が覚めない。

午後になると、2杯目、3杯目を飲んでも集中力が戻らない。

以前は効いていたのに、最近は効かなくなった気がする。

このような状態になると、「カフェインに耐性がついた」「もっと濃いコーヒーを飲めばよい」と考えたくなります。

しかし、コーヒーが効かないように感じる理由は、耐性だけではありません。

睡眠時間が不足している、飲む量や時間帯によって夜の睡眠を悪化させている、いつものカフェインが切れたときの不調をコーヒーで一時的に戻している。このような可能性もあります。

結論から言えば、最初にやるべきことはカフェインを増やすことではありません。

**睡眠時間、1日の摂取量、最後に飲む時刻、飲まなかった日の状態を順番に確認すること**です。

## コーヒーが効かなくなったように感じる5つの理由

主な可能性は次の5つです。

1. 睡眠不足が強く、カフェインだけでは補えない
2. 習慣的な摂取で一部の効果に慣れている
3. 飲む量が増え、夜の睡眠を悪化させている
4. カフェインが切れたときの離脱症状を戻している
5. 眠気の背景に、体調や睡眠障害など別の問題がある

一つに決めつけず、変更しやすい項目から確認します。

## 最初に確認するのは睡眠時間

コーヒーが効かないと感じたとき、最優先で確認したいのは前夜の睡眠です。

カフェインには眠気や注意力低下を一時的に和らげる作用があります。しかし、睡眠不足による影響を完全に消せるわけではありません。

48時間の断眠条件でカフェインを繰り返し投与した試験では、認知・注意機能の低下は軽減されましたが、十分に休息した状態まで完全には戻りませんでした。

つまり、睡眠不足が大きいほど「コーヒーが弱くなった」というより、**眠気の強さがカフェインで補える範囲を超えている**可能性があります。

まずは直近7日間について、次を確認してください。

- 就寝時刻と起床時刻
- 実際に眠れた時間
- 夜中に目が覚めた回数
- 起床時の回復感
- 日中に強い眠気が出た時刻

睡眠時間が普段より短い日ほどコーヒーの追加量が増えているなら、飲み方より先に睡眠時間の確保を見直す必要があります。

仕事中の強い眠気全体を見直したい場合は、HOVELの「仕事中に眠すぎる人が最初に見直すべきこと」も参考にしてください。

## カフェイン耐性は起こり得るが、一律ではない

カフェインを習慣的に摂ると、同じ量で以前と同じ感覚を得にくくなることがあります。これは一般に耐性と呼ばれます。

ただし、すべての効果に同じ速さで完全な耐性が生じるわけではありません。研究では、主観的な覚醒感、血圧、睡眠への影響などで、耐性の出方が異なることが示されています。

そのため、「毎日飲んでいるから完全に効かない」とも、「耐性は存在しない」とも言い切れません。

また、普段カフェインを摂っている人が一定時間カフェインを控えたあとに飲むと、気分や反応速度への効果を強く感じる場合があります。これは純粋な能力向上だけでなく、離脱による不調が解消された影響を含む可能性があります。

耐性が気になる場合でも、自己判断で大量に増やすのは避けてください。

## 1日の量と最後に飲む時刻を確認する

健康な成人について、FDAとEFSAは、1日400mgまでのカフェインを、一般に健康への悪影響と関連しない目安として示しています。

ただし、これは「400mgまでなら誰でも問題ない」という意味ではありません。

カフェインへの感受性や代謝速度には個人差があります。妊娠中、授乳中、服薬中、心疾患や不安症状がある場合などは、より慎重な判断が必要です。

さらに、コーヒー1杯のカフェイン量は、豆、抽出方法、容量によって大きく変わります。コーヒーだけでなく、緑茶、紅茶、エナジードリンク、チョコレート、眠気対策製品、医薬品に含まれるカフェインも合計してください。

### 高用量は早い時間でも睡眠へ影響することがある

2025年掲載のランダム化クロスオーバー試験では、カフェイン400mgを就寝12時間前、8時間前、4時間前に摂取した場合、睡眠開始や睡眠構造などへ影響が確認されました。

一方、同試験では100mgについて、就寝4時間前までの条件で有意な影響は確認されませんでした。ただし、参加者は23人の若年男性であり、結果をすべての人へそのまま当てはめることはできません。

別の試験では、400mgを就寝6時間前に摂った場合でも、総睡眠時間が減少しました。

重要なのは、単純に「午後は全部禁止」と決めることではなく、**量と時刻を一緒に記録すること**です。

例えば22時に寝る人が、16時に大容量コーヒーやエナジードリンクを飲んでいる場合、その日の寝つきや夜間覚醒、翌朝の眠気まで記録します。

自分では眠れているつもりでも、睡眠への影響を正確に自覚できない場合がある点にも注意が必要です。

## 急にやめると、かえって眠くなることがある

毎日カフェインを摂っている人が急に中止すると、次のような離脱症状が出ることがあります。

- 頭痛
- 疲労感
- 眠気
- 集中しにくい
- 気分の落ち込みや irritability

離脱症状は、最後の摂取から12〜24時間ほどで始まり、20〜48時間前後に強くなることがあると報告されています。

そのため、「効かなくなったから明日から完全にやめる」と急に変更すると、一時的に仕事の能率が落ちたように感じる可能性があります。

減らす場合は、体調を見ながら段階的に行います。

例えば、次の順で減らします。

1. まず1日の摂取量を把握する
2. 夕方以降の1杯をデカフェやノンカフェイン飲料へ替える
3. 大容量を小容量へ替える
4. 数日ごとに少しずつ減らす

強い頭痛や体調不良が出る場合は、無理に継続しないでください。

## 1週間で確認する方法

原因を切り分けるには、感覚だけで判断せず、1週間記録するのが有効です。

毎日、次の5項目を書きます。

| 項目 | 記録内容 |
|---|---|
| 睡眠 | 就寝、起床、推定睡眠時間 |
| カフェイン | 飲料名、量、摂取時刻 |
| 最終摂取 | その日最後に飲んだ時刻 |
| 眠気 | 強く眠くなった時刻と程度 |
| 体調 | 頭痛、動悸、不安感、胃の不快感 |

1週間後、次の順で見直します。

### 1. 睡眠時間が短い日に眠気が強くないか

当てはまるなら、カフェイン量を増やすより、就寝を早めるために削る行動を決めます。

### 2. 最後のカフェインが遅い日に翌朝つらくないか

当てはまるなら、最終摂取時刻を少しずつ前倒しします。

### 3. 飲まなかった日に頭痛や強い疲労が出ないか

当てはまるなら、急な中止ではなく段階的な減量を検討します。

### 4. 昼食後だけ眠気が強くないか

昼食後に限って眠くなる場合は、睡眠不足に加えて、昼食量、食べ方、食後の行動も確認します。詳しくは「昼食後に眠くなる原因とは？会社員が午後の集中力を落とさない方法」を参照してください。

## コーヒーを追加する前にできること

眠気が出たときは、カフェインを足す前に次を試します。

- 立って数分歩く
- 明るい場所へ移動する
- 水分を摂る
- 可能なら短時間の仮眠を取る
- 単調な作業から、会話や確認を伴う作業へ切り替える

ただし、これらも睡眠不足そのものを解消する方法ではありません。

眠気を一時的に乗り切る方法と、夜の睡眠を立て直す方法を分けて考えてください。

## 医療機関へ相談した方がよいケース

次のような状態がある場合は、コーヒーだけで対処を続けず、医療機関へ相談してください。

- 十分に寝ているつもりでも、日中に耐え難い眠気が続く
- 会議中や運転中に眠り込みそうになる
- 大きないびき、睡眠中の呼吸停止を指摘された
- 朝の頭痛や強い倦怠感が続く
- 動悸、胸痛、強い不安、手の震えがある
- 眠気のために仕事や生活へ明確な支障が出ている

眠気の背景には、睡眠時無呼吸症候群、睡眠障害、貧血、甲状腺の問題、服薬の影響などが隠れている場合があります。記事だけで原因を診断することはできません。

## まとめ｜効かないと感じたら、量より記録を増やす

コーヒーが効かなくなったように感じても、すぐに量を増やす必要はありません。

確認する順番は次のとおりです。

1. 睡眠時間が不足していないか
2. 1日のカフェイン量が増えていないか
3. 最後に飲む時刻が遅くなっていないか
4. 飲まない日に離脱症状が出ていないか
5. 強い眠気が長期間続いていないか

カフェインは、適切に使えば一時的な覚醒や集中を助けます。しかし、睡眠不足を帳消しにするものではありません。

まず7日間、睡眠時間、摂取量、最終摂取時刻、眠気を記録してください。

増やす前に原因を切り分けることが、翌日の眠気まで含めた改善につながります。
