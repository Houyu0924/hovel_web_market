# HOVEL Repository v2 導入手順

1. このZIPを展開する
2. `install_v2.py` を、クローン済みの `hovel_web_market` フォルダへコピーする
3. ターミナルでリポジトリへ移動する
4. 次を実行する

```bash
python3 install_v2.py
```

5. GitHub Desktopで変更を確認する
6. Summaryに次を入力する

```text
feat: implement HOVEL Repository v2
```

7. `Commit to main` → `Push origin`

既存ファイルと同じパスがある場合は、`.hovel_backup_v2/` にバックアップしてから更新します。
