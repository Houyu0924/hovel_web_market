# HOVEL Knowledge Base

全AIが最初に参照する共通知識基盤。

## Principle
1. 既存情報を検索する
2. 情報の鮮度と根拠レベルを確認する
3. 不足情報だけを追加調査する
4. 新しい情報を標準形式で保存する
5. 記事と根拠の対応を残す

## Directories
- `research/`: 論文、公的資料、学会資料
- `market/`: 市場調査、需要仮説
- `products/`: 商品データ
- `keywords/`: キーワード、検索意図
- `competitors/`: 競合分析
- `experiments/`: HOVEL独自検証
- `personas/`: 読者像
- `glossary/`: 用語定義

## Source Rule
情報には必ず、出典、参照日、根拠レベル、要約、利用可能範囲を記録する。
