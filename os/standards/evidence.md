# Evidence Standard

| Level | Source | Usage |
|---|---|---|
| A | 査読論文、公的機関、学会、法令 | 中心根拠として使用可能 |
| B | メーカー公式、製品仕様書、一次データ | 商品仕様や企業事実に使用可能 |
| C | 信頼できる専門媒体、専門家解説 | 補足情報として使用 |
| D | ECレビュー、SNS、掲示板、動画コメント | 仮説・傾向の把握に限定 |
| E | AI推論、未確認情報 | 公開記事の事実根拠に使用禁止 |

- 数値、効果、比較優位には出典を付ける。
- 商品レビューの傾向を示す場合、個別レビューを一般化しない。
- 相関を因果として書かない。
- 研究対象、条件、限界を省略して過度に一般化しない。
- 参照日を記録する。
