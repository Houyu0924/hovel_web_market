# 制作目的

既存の「会社員向けアイマスク比較」記事を新規量産するのではなく、検索意図、商品情報、比較軸、収益導線を再審査し、更新または統合する。

# 検索意図

仕事中の休憩、自宅睡眠、出張・移動のいずれかで使えるアイマスクを比較し、自分の用途に合う商品を選びたい。

# 読者

25〜30代の男性会社員。睡眠不足や昼休みの休息に悩むが、商品の違いを判断できていない人。

# 既存記事との関係

既存記事のURL、公開日、比較商品、検索流入を確認するまで、新規記事として制作しない。

# 収益仮説

高単価商品だけを優先せず、用途別の選択肢からAmazon等の商品ページへ送客する。初期KPIは売上ではなく商品リンクCTR。

# 商品選定条件

- 公式または販売元で素材・サイズ・洗濯方法を確認できる
- 自宅、昼休み、出張のどの用途に向くか説明できる
- 実測していない遮光性や快適性を断定しない
- 価格帯を分散させる
- 入手性が極端に低い商品を避ける
